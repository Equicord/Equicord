# VimMotions

a plugin that brings **Vim-style motions** to Discord.
switch between *Insert* and *Normal* modes, and control chat scrolling using Vim keybinds.

---

## ‚ú® Features

- `Esc` ‚Üí switch to **Normal mode**
- `i` ‚Üí switch back to **Insert mode**
- `j` ‚Üí scroll **down**
- `k` ‚Üí scroll **up**
- `0` ‚Üí scroll to **top**
- `$` ‚Üí scroll to **bottom**

status bar shows the current mode:
- **-- NORMAL --** (in red)
- **-- INSERT --** (in green)
