# Better Forwards

## Resend on fail

Automatically resends a forwarded message as text if the forward fails. Can be used to bypass channel restrictions.

## Forward Preface

The prefix that's used for the "Forwarded from" text when Resend on fail is enabled.

## Dont Follow Forwards

Don't follow forwarded messages after sending them.
